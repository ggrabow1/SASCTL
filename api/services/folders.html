sasctl.services.folders¶

class sasctl._services.folders.Folders[source]¶

Bases: Service

The Folders API provides an organizational structure for SAS and external content. It can also be used for favorites folders or a history of objects accessed. The resources that are stored in folders (members) use a URI to point back to those resources.

classmethod get_folder(item, refresh=False)¶

Return a folder instance.

Parameters:

  • item (str or dict) – Name, ID, or dictionary representation of the folder.

  • refresh (bool, optional) – Obtain an updated copy of the folder.

Returns:

A dictionary containing the folder attributes or None.

Return type:

RestObj or None

Notes

If item is a complete representation of the folder it will be returned unless refresh is set. This prevents unnecessary REST calls when data is already available on the client.
